Telegram Bot for Product Sales (E-commerce)

Developed a Telegram bot to automate product sales (using badminton shuttlecocks as an example).

Features:

Product catalog with wholesale and retail pricing
Order placement with quantity selection
Automatic submission of orders to a Telegram group with customer data
User activity tracking (clicks, navigation, interactions)
Integration with Google Sheets for product and price management

Results:

Simplified ordering process for customers
Automated request handling
Ability to track user behavior

Technologies:
Python, Telegram Bot API, Google Sheets API, file handling
